Visual Studio MVVM Project Template
------ ------ ---- ------- --------

Instructions:

1) Install MvvmTemplateWizard into the GAC using "gacutil /i MvvmTemplateWizard.dll" from the command line (will require admin rights)
2) Copy MvvmTemplate.zip to your "Documents\Visual Studio 2010\Templates\ProjectTemplates\Visual C#" folder
3) Start Visual Studio 2010 and select New Project
4) Select "Visual C#" from the tree on the right (it should be the root node) and then navigate to the "Model-View-ViewModel Template Project"
5) When the wizard appears, tailor the project template to your project's needs
